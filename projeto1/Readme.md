# 📊 Dashboard de Vendas em Excel

Dashboard interativo desenvolvido em **Microsoft Excel** para análise de desempenho de vendas.

O projeto apresenta indicadores estratégicos (KPIs) e visualizações que permitem acompanhar o desempenho comercial ao longo do tempo, facilitando a interpretação de dados e apoiando a tomada de decisão.

---

# 📈 KPIs do Dashboard

O painel apresenta os principais indicadores de desempenho:

* 💰 **Faturamento Total**
* 📦 **Total de Caixas Vendidas**
* 📊 **Ticket Médio de Vendas por Mês**
* 📅 **Análise de vendas por mês**
* 🛒 **Produtos mais vendidos**
* 🌎 **Distribuição de vendas por país**

---

# 📊 Análises disponíveis no dashboard

O dashboard permite visualizar:

* Evolução do faturamento ao longo dos meses
* Comparação de vendas entre anos
* Ranking de produtos mais vendidos
* Participação de vendas por país
* Ticket médio mensal
* Indicadores consolidados de desempenho

Também conta com **segmentações (filtros)** para análise dinâmica por:

* Ano
* Mês

---

# 🧰 Ferramentas utilizadas

* Microsoft Excel
* Tabelas Dinâmicas
* Gráficos Dinâmicos
* Segmentação de Dados (Slicers)
* KPIs e indicadores de performance
* Modelagem e organização de dados

# 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar **análise de dados e construção de dashboards**, transformando dados brutos em indicadores visuais que ajudam a identificar padrões, tendências e oportunidades de negócio.
